# LLMs, RAG & Agentic AI

A comprehensive 3-day course covering Large Language Models, Retrieval-Augmented Generation, Agentic AI, and Enterprise AI Architecture.

## Course Structure

### Day 1 – Foundations of LLMs and RAG
1. Foundations of Generative AI
2. The LLM Training Lifecycle
3. Prompt Engineering and Context Engineering
4. Prompt Management at Scale
5. LLMs and the AI Ecosystem
6. RAG Fundamentals and Document Ingestion
7. Embeddings, Vector Databases, and Retrieval

### Day 2 – Building AI Applications
8. Context Processing and Response Generation
9. Evaluation and Testing
10. Memory and State Management
11. Agentic AI Fundamentals
12. Designing Intelligent Agents
13. Tool Calling, Agent Communication, and Multi-Agent Systems
14. AI Frameworks and Development Ecosystem

### Day 3 – Advanced AI and Enterprise
15. Advanced Retrieval and Modern RAG
16. AI Applications: Streaming and Real-time
17. Responsible AI, Security, and Governance
18. AI Cost Engineering
19. Enterprise AI Architecture and Operations

## Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under **Source**, select **GitHub Actions**
4. The site will automatically deploy on every push to `main`

Your site will be available at: `https://your-username.github.io/probable-memory/`
