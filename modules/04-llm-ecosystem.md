## Overview

Become familiar with today's leading language models and the broader AI ecosystem, including embedding models, inference providers, and major frameworks.

## Learning Objectives

- Compare leading LLMs across key dimensions
- Understand the tradeoffs between different model providers
- Navigate the broader AI tooling ecosystem

## Key Topics

### Leading Language Models

| Model | Provider | Strengths |
|:------|:---------|:----------|
| GPT-4o / o1 / o3 | OpenAI | Reasoning, tool use, enterprise support |
| Claude 3.5 / Claude 4 | Anthropic | Safety, long context, coding |
| Gemini 1.5 / 2.0 | Google | Multimodal, long context |
| Llama 3.x | Meta | Open source, community |
| DeepSeek V3 / R1 | DeepSeek | Cost-effective, strong reasoning |
| Qwen 2.5 | Alibaba | Multilingual, open source |
| Mistral / Mixtral | Mistral | European, efficient |
| Phi-4 | Microsoft | Small but capable |

### Comparison Dimensions
- Reasoning ability
- Context window size
- Latency
- Cost per token
- Multimodal capabilities
- Enterprise suitability
- Open source vs. proprietary

### The Broader Ecosystem
- Embedding models (OpenAI, Cohere, open-source)
- Inference providers (Together, Fireworks, Groq, Replicate)
- Model hosting platforms (Hugging Face, Ollama, vLLM)
- Frameworks overview (preview of Module 11)

## Discussion Questions

- What factors matter most when choosing an LLM for production?
- When would you choose an open-source model over a proprietary one?
- How does multimodal capability change application design?
